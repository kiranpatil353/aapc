== Description ==
Auto assign post category based on tags mapped by user in plugin setting. Single tag can be mapped to multiple categories. Users need to map tags with available categories first. 


= A Few Feature Highlights =
* Single tags can be mapped to multiple categories 
* Useful for post authors to assign multiple categories automatically.
* Easy to use/customize
* Free to use anywhere
* Even you update the existing posts it will assign categories.

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: Auto Assign Post Category ::
    Press Install Now for the Show notice or message on admin area plugin
    Press Activate Plugin
	then assign tags to categories availbale in your website

To install manually instead:

    Upload the Auto-Assign-Post-Category  directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
	then assign tags to categories availbale in your website
